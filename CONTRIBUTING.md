# Contributing to nepali-address

## Getting Started
1. Fork the repository
2. Clone your fork: 
```bash
git clone https://github.com/holasoymas/nepali-address.git
```
3. Install dependencies: 
```bash
npm install
```

## Development
- Run tests:
```bash
npm test
```
- Build project: 
```bash
npm run build
```
- Generate docs: 
```bash
npm run docs
```

## Submitting Changes
1. Create a new branch: 
```bash
git checkout -b feature/your-feature
```
2. Make your changes

3. Run tests to make sure nothing breaks: 
```bash
npm test
```

4. Commit your changes: 
```bash
git commit -m "feat: your feature"
```

5. Push and open a Pull Request

## Reporting Bugs
Open an issue on GitHub with:
- What you expected to happen
- What actually happened
- Steps to reproduce
